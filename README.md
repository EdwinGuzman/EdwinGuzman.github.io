# Blog

Personal blog based on a static blog generator and starter gatsby repo. A port of [Casper](https://github.com/TryGhost/Casper) v3 a theme from [Ghost](https://ghost.org/) for [GatsbyJS](https://www.gatsbyjs.org/) using [TypeScript](https://www.typescriptlang.org/).

Run `gatsby develop` to work locally.

Run `npm run deploy` to deploy to Github pages.
